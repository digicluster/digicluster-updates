# DigiCluster Updates

This repository contains official over-the-air (OTA) software updates for DigiCluster.

OTA updates allow your DigiCluster unit to receive improvements without needing to remove the unit from the vehicle or manually reinstall software.

Updates may include:

- New features
- Performance improvements
- Bug fixes
- Compatibility updates

To help ensure a smooth update process, only install official DigiCluster updates and follow any included update instructions carefully.

For more information, visit the main DigiCluster documentation and support resources.

To report any bugs/issues, fill out the contact form on our website: DigiCluster.org.
Alternatively, you can reach out in our Discord: https://discord.gg/WGhnSSF6HS
